# Devices

This directory contains security assessments of various IoT devices across multiple categories, including cameras, firewalls, routers, smart home products, and industrial systems.

Each assessment follows a structured methodology and may include:

- Device identification and reconnaissance
- Hardware analysis
- PCB inspection
- UART, JTAG, SPI, and eMMC analysis
- Firmware acquisition and analysis
- Bootloader analysis
- Operating system identification
- Network service enumeration
- Attack surface mapping
- Vulnerability research
- Security findings and recommendations

The goal of this directory is to document the security posture of embedded devices while providing a repeatable methodology for IoT security assessments.
